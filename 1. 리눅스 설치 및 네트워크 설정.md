# 리눅스 설치 및 네트워크 설정

### 1. VM 생성 및 기본 사양 설정

새로 만들기(New)로 VM을 생성한다.

- 이름: oracle19c_class1
- 종류: Linux
- 버전: Oracle Linux (64-bit)

메모리 8GB, 프로세서 4개로 설정한다.

디스크는 50GB로 설정한다.

속성에서 포인팅 장치를 USB 태블릿으로 변경한다.

> 포인팅 장치를 USB 태블릿으로 설정하면 VM 내 마우스 동작이 안정적으로 동작한다.

### 2. OS 설치 (언어/소프트웨어/계정)

설치(Install)를 시작한다.

언어는 한국어로 선택한다.

소프트웨어 선택에서 아래와 같이 설정한다.

- 기본 환경: 서버 (GUI 포함)
- 추가 기능: 개발 도구, 시스템 관리 도구

네트워크 및 호스트명을 설정한다.

- 호스트명: orcl

root 암호를 설정하고 oracle 유저를 생성한다.

설치 완료 후 라이선스 정보에 동의한다.

---

### 3. VirtualBox 네트워크 브리지 설정

가상머신이 호스트 PC와 동일한 네트워크 대역을 사용하도록 설정한다.

```
VM 선택 → 설정 → 네트워크 → 어댑터 1
다음에 연결됨: 어댑터에 브리지(Bridged Adapter)
```

> 브리지 모드로 설정해야 호스트 PC와 동일한 네트워크 대역의 IP를 사용할 수 있다.

---

### 4. Linux VM 고정 IP 설정

VM 내부에서 고정 IP를 설정한다.

| 항목 | 값 |
|------|-----|
| IP 주소 | 192.168.13.181 |
| 넷마스크 | 255.255.255.0 |
| 게이트웨이 | 192.168.13.1 |
| DNS | 8.8.8.8 |

> IP는 윈도우 IP + 50번으로 설정하는 것을 권장한다.

네트워크 서비스를 재시작한다.

```bash
systemctl restart NetworkManager
```

---

### 5. 네트워크 설정 확인

IP가 정상 적용됐는지 확인한다.

```bash
# IP 확인
ip a

# 게이트웨이 확인
ip route

# 외부 통신 확인
ping -c 3 8.8.8.8
```

---

### 6. PuTTY / MobaXterm 접속

호스트 PC에서 아래 정보로 SSH 접속한다.

| 항목 | 값 |
|------|-----|
| Host | 192.168.13.181 |
| Port | 22 |
| Connection type | SSH |

접속이 안 되는 경우 아래 항목을 순서대로 확인한다.

1. VM IP 설정 확인 (`ip a`)
2. 브리지 어댑터 설정 여부 확인
3. 방화벽 및 sshd 상태 확인

```bash
systemctl status sshd
firewall-cmd --list-all
```

MobaXterm으로 접속 후 ping이 정상인지 확인한다.

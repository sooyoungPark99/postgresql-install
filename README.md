# PostgreSQL 설치 및 Oracle 19c 환경 구축

PostgreSQL 15 설치 및 Oracle 19c Single Instance 환경 구축 가이드입니다.
VirtualBox 기반의 실습 환경에서 직접 설치하며 작성한 문서입니다.

---

## 사용 소프트웨어

| 소프트웨어 | 버전 |
|-----------|------|
| VirtualBox | 7.0.18 |
| Oracle Linux | release 7.9 |
| PostgreSQL | 15 |
| Oracle Database | 19c (19.3.0) |
